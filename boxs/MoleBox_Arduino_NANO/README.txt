# Summary

<p>MoleBox system - 3D printed boxes compatible with BITBEAM construction kit (therefore even with LEGO Technics/Mindstorms). The main purpose of MoleBox is covering a particular sensor or the entire module and simultaneously allowing variable fixing sensors/modules. MoleBoxes are suitable for measurements with Arduino (<a href="http://www.e-mole.cz/diy/molegraph">MoleGraph</a>, school measuring system based on Arduino) or even as part of a robotic construction (<a href="http://www.e-mole.cz/diy/m-bitbeam">m-BITBEAM</a>).</p>
<p>Box for Arduino NANO.</p>
<p>More info on <a target="_blank" href="http://www.e-mole.cz/diy/molebox">e-Mole MoleBox page</a> (czech only).</p>
