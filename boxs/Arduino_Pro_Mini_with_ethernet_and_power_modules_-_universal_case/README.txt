# Summary

Universal case for Arduino Projects. Up to 4 modules:
- power supply module (23.5mm x 51.5mm)
- Arduino Pro Mini
- w5100 ethernet module
- one additional sensor/module (up to 34mm x 31mm)

Used for home automation projects. Can be used as well with step-down dc-dc module.

Printed with Devil Design's PETG.
