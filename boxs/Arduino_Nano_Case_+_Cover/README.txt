# Summary

This case is designed to contain the Arduino Nano board and to allow additional free space for wiring, other components etc..

There are 3 openings:
1 - Arduino USB Connector
2 - DC Female Conector
3 - Hole for wiring to come out of the box

I think this case can be used as general Nano box project as it provides additional space and power in option.

Together with the case I used:
DC female connector - http://bit.ly/2xlHA2J
5v 6A Power supply - http://bit.ly/2xlTLfM

The cover that is attached to this project will seal the box completely but it's possible to design a cover that will allow opening for passing wires when in beta testing stage.
